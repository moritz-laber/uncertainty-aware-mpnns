# Robustness and Generalization in Uncertainty-Aware Message Passing Neural Networks

This repository contains code for the paper *Robustness and Generalization in Uncertainty-Aware Message Passing Neural Networks* (2026) by Alesia Chernikova, Moritz Laber, Narayan G. Sabhahit and Tina Eliassi-Rad, which will be published in the proceedings of AISTATS 2026.
