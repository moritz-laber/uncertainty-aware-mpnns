# Robustness and Generalization in Uncertainty-Aware Message Passing Neural Networks

This repository contains code for the paper [*Robustness and Generalization in Uncertainty-Aware Message Passing Neural Networks*](https://openreview.net/forum?id=BcqtGTw9OZ) by Alesia Chernikova, Moritz Laber, Narayan G. Sabhahit and Tina Eliassi-Rad.

You can find the paper with appendices in this repository as `robustness_and_generalization_in_uncertainty-aware_message_passing_neural_networks.pdf`.
